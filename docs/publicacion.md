# Publicacion por enlace web

Esta version funciona como sitio web estatico y puede publicarse para que terceros ingresen solo con un enlace, sin copiar archivos a sus computadoras.

## Opciones de publicacion
- Netlify: subir la carpeta completa del proyecto.
- Vercel: desplegar como sitio estatico.
- Hosting compartido con cPanel: copiar `index.html`, `styles.css`, `app.js` y `enhancements.js`.
- Servidor propio con Nginx o Apache.

## Archivos minimos a publicar
- `index.html`
- `styles.css`
- `app.js`
- `enhancements.js`

## Credenciales demo incluidas
- `admin / admin123`
- `direccion / direccion123`
- `cvega / docente123`
- `atorres / docente123`
- `pmedina / docente123`

## Importante
Para una publicacion real todavia hace falta un backend para autenticacion segura, base de datos centralizada y control de sesiones del lado servidor. Esta base ya deja lista la interfaz y la logica de prototipo para pasar a esa etapa.
